# 주의할 점

Firebase 데이터베이스 사용 시 주의할 점은 다음과 같습니다.

  1. 키(key) 값에 슬래시(/)가 포함되서는 안됩니다.
  2. set(), update() 메서드를 구분해 사용해야 합니다.
  3. on 이벤트는 필요한 경우만 사용해야 합니다. (지속적인 이벤트 처리가 불필요한 경우는 once 활용)
  4. Firebase 데이터베이스 읽기는 비동기로 작동합니다. (동기 처리가 필요한 경우 async 함수 활용)
  5. 효율적인 데이터베이스 관리를 위해 평면화 하여야 합니다.
  6. 정렬과 필터링을 적재적소에서 활용합니다.
