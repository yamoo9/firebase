# 프로젝트 관리

Google 클라우드 콘솔 매니저에 접속하여 프로젝트를 제거하거나, 제거 예정 프로젝트를 복원할 수 있습니다.

# 제거

프로젝트 목록에서 제거할 프로젝트를 선택 후, 삭제 버튼을 누릅니다.

주의!

삭제된 프로젝트는 삭제 대기 중인 리소스에서 대기(복원 가능) 상태로 있다가 1주일이 지나면 완전 삭제 됩니다.

# 복구

삭제 대기 중인 리소스 페이지에서 복구할 프로젝트를 선택 후, 복원 버튼을 누릅니다.
