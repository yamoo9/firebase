# 스토리지 생성

Firebase 콘솔 > Storage를 선택한 후, 시작하기 버튼을 누릅니다.

보안 규칙 팝업 창이 뜹니다. 보안 규칙은 파일을 저장하고 접근 제어를 설정하는 방법에 대한 규칙을 정의 할 수 있습니다. 파일은 Google Cloud Storage에 저장됩니다. 보안 규칙은 나중에 변경 가능하니 확인 버튼을 누릅니다.

# 폴더 생성

스토리지가 생성되면 폴더를 만들거나, 파일을 업로드 할 수 있습니다. 폴더 생성 버튼을 클릭해 images 폴더를 생성해봅니다.

# 파일 업로드

생성된 폴더를 클릭해 이동한 다음 업로드 버튼을 눌러 이미지 파일을 업로드 해봅니다.

# 다운로드 URL과 Token

업로드한 이미지 파일을 클릭하면 파일 정보와 파일 위치 정보를 확인할 수 있습니다.

다운로드 URL 주소를 클릭하면 다운로드 가능한 주소가 클립보드에 복사됩니다. token 정보가 주소에 포함된 것에서 알 수 있듯이 인증이 허가된 사용자가 아닐 경우 이미지를 보거나 다운로드 할 수 없습니다.

  • token 정보가 포함된 다운로드 URL
  • token 정보가 포함되지 않은 다운로드 URL
