# 로그인

Firebase 웹 사이트에 접속한 후, 로그인 링크를 눌러 가입을 진행합니다.

Google에 등록된 계정이 없다면 계정 만들기 링크를, 계정이 있다면 이메일을 입력한 후, 다음 버튼을 누릅니다.

등록된 계정 비밀번호를 입력한 후, 다음 버튼을 눌러 진행합니다.

# 콘솔

로그인에 성공하면 콘솔로 이동 링크를 눌러 콘솔 페이지로 이동합니다.

콘솔 페이지가 표시되면 최근에 진행한 프로젝트가 리스트 되어 있고, 새로운 프로젝트를 추가 할 수도 있습니다.

# 프로젝트 추가

프로젝트 추가 버튼을 누르면 프로젝트 추가 모달 창이 화면에 표시됩니다. 프로젝트 이름, ID, 애널리틱스 위치, Cloud Firestore 위치 등을 기입, 선택한 후 약관에 동의하고 프로젝트 만들기 버튼을 눌러 프로젝트를 생성합니다.

신규 프로젝트 생성에 성공하면 다음 화면이 표시됩니다.

# Firebase CLI 도구

Firebase (CLI) Tools 패키지를 글로벌 설치합니다.

 


$ yarn global add firebase-tools  #  또는 npm i firebase-tools -g
$ firebase -V                     # firebase-tools 버전 정보 출력

# CLI 로그인

도구 설치가 완료되면 명령창에 firebase login을 입력 실행하여 로그인을 시도합니다. 사용자의 기록 및 정보 수집을 요청하는 질문에는 아니오(No)로 답변하고 넘어갑니다.

 




$ firebase login

# Firebase가 익명으로 사용자의 CLI 사용 및 오류보고 정보를 수집하도록 허용하시겠습니까? 아니오
? Allow Firebase to collect anonymous CLI usage and error reporting information? No

웹 브라우저에 로그인 창이 표시되면 내용을 읽고 허용 버튼을 눌러 로그인 합니다.

로그인이 성공하면 화면에 Firebase CLI Login Successful 메시지가 표시됩니다.

명령창에 다시 firebase login 명령을 실행하면 명령창 화면에 로그인 되었다고 안내해줍니다.

 



$ firebase login
