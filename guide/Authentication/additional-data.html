# 인증 사용자 정보 추가

인증된 사용자 정보에 데이터를 추가하려면 Firebase 데이터베이스를 활용해 인증 사용자 객체에 새로운 데이터를 추가할 수 있습니다.

NOTE

Firebase 데이터베이스 사용법을 공부한 후 코드를 살펴보세요.

사용자의 "최애도서"를 입력받아 사용자 정보에 추가해보겠습니다. 먼저 입력받을 폼 템플릿을 작성합니다. 폼 전송을 처리할 updateUserCustomData 메서드를 연결하고, 인풋 컨트롤에 v-model 디렉티브를 사용해 likeBook 데이터를 바인딩 합니다.

 



 




<form @submit.prevent="updateUserCustomData">
  <h3>나의 최애도서는?</h3>
  <div class="form-control">
    <label for="user-custom-data">최애도서</label>
    <input type="text" id="user-custom-data" v-model="likeBook">
  </div>
  <button>업데이트</button>
</form>

Firebase 데이터베이스 모듈을 불러온 후, likeBook 속성을 컴포넌트 데이터에 추가합니다.

 



 



import { auth, database } from 'firebase';

export default {
  data: () => ({
    likeBook: ''
  }),
}

updateUserCustomData 메서드를 정의한 다음 인증 사용자 객체를 참조합니다. 인증 사용자가 있다면 Firebase 데이터베이스에서 users 참조를 가져와 자식 참조로 인증 사용자의 uid 데이터에 likeBook 속성을 컴포넌트 likeBook 데이터 값으로 업데이트 합니다.





 
 
 




methods() {
  updateUserCustomData() {
    const authUser = auth().currentUser;
    if (authUser) {
      database().ref('users').child(authUser.uid).update({
        likeBook: this.likeBook
      });
    }
  }
}

업데이트 된 데이터를 인증된 사용자의 정보 데이터로 활용하려면 컴포넌트 생성 시점 인증 상태 변경을 감지하는 콜백 함수 내부에 Firebase 데이터베이스 users > user.uid에서 likeBook 값을 가져와 컴포넌트의 likeBook 데이터와 authUser 데이터의 likeBook 속성으로 할당해야 합니다.






 
 
 
 
 
 
 
 




created() {
  auth().onAuthStateChanged(user => {
    this.authUser = user;
    if (user) {
      // ...
      database().ref('users').equalTo(user.uid)
        .once('value', snapshot => {
          const snapVal = snapshot.val();
          if (snapVal) {
            this.likeBook = snapVal.likeBook;
            Vue.set(this.authUser, 'likeBook', this.likeBook);
          }
        });
    }
  });
},
