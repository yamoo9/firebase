# 이메일/패스워드 인증

Firebase 인증 방법 중 이메일과 패스워드를 사용하는 방법을 학습합니다.

# Firebase 로그인 방법 설정

Firebase 프로젝트는 이메일/패스워드 또는 외부 공급자(예: Google, Facebook, GitHub 등) 인증 방법을 제공합니다. 사용할 인증 방법을 활성화 하려면 Firebase 프로젝트 페이지에서 Authentication > 로그인 방법을 선택합니다. 이메일/비밀번호 방법으로 로그인할 수 있도록 설정하기 위해 이메일/비밀번호 상태를 '사용 설정'으로 변경 저장합니다.

# Vue 컴포넌트

LogIn 예제 파일을 압축을 푼 후, Vue 컴포넌트의 템플릿으로 코드를 옮깁니다.

주의!

예제 코드 중 실습과 관련된 코드만 아래 게시되었습니다.





 




 



 
 






 
 
 
 




<template>
  <form @submit.prevent>
    <!-- 이메일 입력필드 ... -->
    <input
      v-model="email"
      type="email"
    >
    <!-- 패스워드 입력필드 ... -->
    <input
      v-model="password"
      type="password"
    >
    <!-- 가입/로그인 버튼 ... -->
    <button v-if="mode === 'signUp'" @click="signUp">가입</button>
    <button v-else @click="signIn">로그인</button>
  </form>
</template>

<script>
export default {
  data:() => ({
    mode: 'signUp',
    email: '',
    password: '',
    authUser: null
  })
}
</script>

# 가입

Firebase 모듈에서 auth를 불러온 후, createUserWithEmailAndPassword 메서드를 사용해 사용자가 입력한 이메일과 패스워드를 사용해 가입하는 코드를 작성합니다. 오류가 발생할 경우, 콘솔에 오류 메시지를 출력하는 코드를 catch 구문에 작성합니다.

 




 
 
 
 
 




import { auth } from 'firebase';

export default {
  // ...
  methods: {
    signUp() {
      const { email, password } = this;
      auth().createUserWithEmailAndPassword(email, password)
        .catch(error => console.error(error.message));
    },
    // ...
  }
}

# 로그인

이미 가입된 사용자 계정을 처리하는 signInWithEmailAndPassword 메서드를 사용해 사용자가 입력한 이메일과 패스워드로 로그인을 시도하는 코드를 작성합니다. 오류가 발생할 경우, 콘솔에 오류 메시지를 출력하는 코드를 catch 구문에 작성합니다.





 
 
 
 
 




export default {
  // ...
  methods: {
    // ...
    signIn() {
      const { email, password } = this;
      auth().signInWithEmailAndPassword(email, password)
        .catch(error => console.error(error.message));
    },
    // ...
  }
}

# 인증 상태 감지

Firebase 가입 또는 로그인의 인증 변경 상태를 감지하여 인증된 사용자 정보를 업데이트하는 코드를 작성합니다. onAuthStateChanged 메서드는 인증 상태 변경에 따라 콜백함수의 첫번째 인자로 사용자 정보를 전달합니다.




 
 
 
 
 






export default {
  // ...
  created() {
    auth().onAuthStateChanged(user => {
      if (user) {
        this.authUser = user;
      } else {
        this.authUser = null;
      }
    });
  },
  // ...
}
