# 실습 준비

Firebase를 활용한 애플리케이션 인증(Authentication)에 대해 알아봅니다.

NOTE

생성된 Firebase 프로젝트가 없을 경우, Firebase 콘솔에서 프로젝트 추가한 후 진행합니다.

로그인 폼 실습 예제로 LoginForm.zip 파일을 다운로드 받아 압축을 풉니다.

# 프로젝트 생성

Vue.js프레임워크를 활용한 프로젝트를 생성합니다.

$ vue create <프로젝트_이름>
1

주의!

Vue 프로젝트를 생성하는 명령을 사용하려면 Vue CLI가 설치되어 있어야 합니다.

# Firebase SDK 설치

생성한 프로젝트에 Firebase SDK 모듈을 설치합니다.

$ yarn add firebase --dev  #  또는 npm i firebase -D
1

참고자료

자바스크립트 프로젝트에 Firebase 추가 문서를 참고하세요.

# 앱에 Firebase 추가

Firebase 콘솔 페이지의 메인 영역에서 웹 앱을 Firebase 추가하기 버튼을 클릭합니다.

Firebase 애플리케이션을 초기화 하는 설정 코드를 복사합니다.

  1. auth/index.js 파일을 만든 후 Firebase SDK 모듈을 불러옵니다.
  2. 앞서 복사한 코드를 붙여넣고 환경설정 상수 config를 정의합니다.
  3. 애플리케이션 초기화 메서드에 환경설정 상수 config를 전달하는 코드를 작성합니다.


 

 
 
 
 
 
 
 
 

 

// auth/index.js

import firebase from 'firebase'; // [1]

const config = { // [2]
  apiKey: "AIzaSyDUTmyoIJousgR_ChVXuJK_xNBeJvWvY_U", // <API_KEY>
  authDomain: "fir-auth-with-vue.firebaseapp.com", // <PROJECT_ID>
  databaseURL: "https://fir-auth-with-vue.firebaseio.com", // <DATABASE_NAME>
  projectId: "fir-auth-with-vue",
  storageBucket: "fir-auth-with-vue.appspot.com", // <BUCKET>
  messagingSenderId: "472331820685"
};

firebae.initializeApp(config); // [3]
